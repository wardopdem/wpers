WPERS MINOR MODE
====================

Description
-----------

This mode can be useful for those who do not like "jumping" cursor.

Install 
-------

Simple add wpers.el folder location to load-path and call `require`:

    (add-to-list 'load-path <path-to-wpers>)
    (require 'wpers)

Using
-----

Toggle mode: 

    `M-x wpers-mode`
